# Introduction to Data Science

## Apa itu Data Science?
**Data Science** adalah bidang yang menggabungkan statistik, pemrograman, dan pemahaman bisnis untuk mengubah data menjadi insight yang berguna dan dapat ditindaklanjuti.

Tujuan utama data science bukan hanya mengolah data, tetapi **membantu pengambilan keputusan berbasis data (data-driven decision making)**.

Secara sederhana:

Data mentah ➝ Diolah ➝ Dianalisis ➝ Dapat insight ➝ Dibuat keputusan

Data science biasanya melibatkan:
- Pengumpulan data
- Pembersihan data
- Analisis eksploratif
- Pembuatan model (machine learning)
- Interpretasi hasil

## Contoh Hasil Kerja Data Science
Hasil kerja data science bisa berupa **insight** maupun **model prediktif**.

**A. Insight (Analytical Result)**

Contoh:
- Mengetahui produk mana yang paling sering dibeli pelanggan
- Menemukan pola waktu pembelian tertinggi
- Mengidentifikasi faktor utama penyebab customer churn

**Jenis analisis yang umum digunakan:**

**1. Descriptive Analytics**
- Menjawab: Apa yang terjadi?
- Contoh: Total penjualan bulan ini

**2. Diagnostic Analytics**
- Menjawab: Mengapa hal itu terjadi?
- Contoh: Penjualan turun karena trafik website menurun

**3. Predictive Analytics**
- Menjawab: Apa yang kemungkinan akan terjadi?
- Contoh: Prediksi penjualan bulan depan

**4. Prescriptive Analytics**
- Menjawab: Apa yang sebaiknya dilakukan?
- Contoh: Rekomendasi strategi diskon untuk meningkatkan penjualan

**B. Modelling (Machine Learning Result)**
Beberapa contoh model dasar dalam data science:

**1. Classification**
- Memprediksi kategori
- Contoh: Apakah pelanggan akan churn atau tidak

**2. Regression**
- Memprediksi angka
- Contoh: Prediksi harga rumah

**3. Clustering**
- Mengelompokkan data tanpa label
- Contoh: Segmentasi pelanggan

**4. Forecasting**
- Memprediksi nilai berdasarkan waktu
- Contoh: Prediksi penjualan 3 bulan ke depan

## Skill yang Dibutuhkan Data Scientist
Seorang data scientist membutuhkan kombinasi beberapa skill berikut:

**1. Statistik & Matematika**
- Probability
- Distribusi data
- Hypothesis testing
- Regression
- Evaluasi model

**2. Programming**
Bahasa yang umum digunakan:
- Python
- SQL
- R (opsional)

Kemampuan penting:
- Data manipulation
- Data cleaning
- Exploratory Data Analysis (EDA)
- Machine learning implementation

**3. Data Visualization**
- Membuat grafik yang informatif
- Menjelaskan insight dengan jelas
- Tools: Matplotlib, Seaborn, Tableau, Power BI

**4. Business Understanding**
- Memahami problem bisnis
- Menerjemahkan kebutuhan bisnis ke dalam problem analitik
- Mengkomunikasikan hasil ke stakeholder

**5. Communication Skill**
- Menjelaskan hasil analisis dengan sederhana
- Storytelling menggunakan data

## Data Science Methodology
Data science tidak dilakukan secara acak. Ada tahapan sistematis yang biasanya diikuti.

Salah satu alur umum adalah:

**1. Business Understanding**  
Memahami masalah bisnis yang ingin diselesaikan.  
Contoh: “Bagaimana cara mengurangi customer churn sebesar 10%?”

**2. Analytic Approach**  
Menentukan pendekatan analitik yang tepat.  
Apakah ini problem klasifikasi? regresi? forecasting?

**3. Data Requirements**  
Menentukan data apa saja yang dibutuhkan.  
Contoh: data transaksi, data demografi pelanggan, histori penggunaan.

**4. Data Collection**  
Mengumpulkan data dari berbagai sumber:
- Database
- API
- File CSV/Excel
- Web scraping

**5. Data Preparation**
Tahap yang paling memakan waktu.
Meliputi:
- Cleaning data
- Handling missing values
- Feature engineering
- Encoding

**6. Exploratory Data Analysis (EDA)**  
Menganalisis pola dan karakteristik data menggunakan statistik dan visualisasi.

**7. Modelling**
Membangun model machine learning sesuai kebutuhan.

**8. Evaluation**  
Mengukur performa model menggunakan metrik seperti:
- Accuracy
- Precision & Recall
- RMSE, dll

**9. Deployment (Opsional)**  
Mengimplementasikan model ke sistem produksi agar bisa digunakan secara real-time.
